## Middle Of the Linked List
[LeetCode Link](https://leetcode.com/problems/middle-of-the-linked-list/)

```cpp
    ListNode* middleNode(ListNode* head) {
        if(head==nullptr){
            return nullptr;
        }
        ListNode* slow=head;
        ListNode* fast=head;
        while(fast!=nullptr&&fast->next!=nullptr){
            slow=slow->next;
            fast=fast->next->next;
        }

        return slow;
    }
```
